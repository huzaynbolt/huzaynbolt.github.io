About Strategic/Program Proposals

Partners who are willing to offer other growth opportunity programs different from trainings are required to provide the following details of their program to help Corps members assess the Growth Opportunity Program they want to offer. NB. A growth opportunity program could be an access to finance program (loans, grants or equity contribution), competition, scholarship, or micro-enterprise opportunity.

How to submit a Strategic/Program Proposals

Partners who are willing to offer other growth opportunity programs different from trainings are required to provide the following details of their program to help Corps members assess the Growth Opportunity Program they want to offer. NB. A growth opportunity program could be an access to finance program (loans, grants or equity contribution), competition, scholarship, or micro-enterprise opportunity.

1). About Program

In this section, you will be required to provide the necessary information about the Impact Program and what the corps member stand to gain by participating.

Details to be provided include:

  1. Select categories/ sub-categories
  2. Program Details
    • Program title
    • Program Type (Loan Finance, Grant Finance, Equity Finance, Micro-enterprise opportunity, competition, scholarship)
    • Program Description (what this program is about, including its goals and objectives)
    • Audience (Who is the program meant for?)
    • Eligibility Criteria for participation

2). Program Packages

A program package contains more specific details about the various iterations of the program that is available, or allows you offer slight variations of the program in different locations as the case may be. If you program has only one iteration, please create only only one package for the program.

Please provide the following information for program package.

  1. Package title.
  2. Package description
  3. Package terms
  4. Start date & duration
  5. Location(s) where this package is applicable
    • Address
    • Local government
    • State
  6. Package Fee(amount):
    • Amount
    • Free (Select “Free” if you are not charging any fees).
  7. What does fee cover? (E.g. course materials, lunch, certificates etc.)
  8. Enrollment deadline.

3). Accreditation information

The SAED Administrator needs to assess your capacity to organize this impact program. Please provide all the necessary information required that will help us adequately assess your competence.

  1. Have you organized this impact Program or any other before?
  2. What qualifies you to organize this program?
  3. Where applicable, what special resources/tools do you possess that will enable you adequately organize this program?
